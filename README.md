# Purdue Research Matchmaker

A research discovery platform built with Python and Flask that helps students explore Purdue faculty, research areas, and potential undergraduate research opportunities.

## Overview

Finding research opportunities can be overwhelming when information is spread across multiple department websites. Purdue Research Matchmaker provides a centralized way to explore faculty interests, discover research domains, and identify professors whose work aligns with a student's academic interests.

The project was built as a personal learning project while exploring undergraduate research opportunities and developing full-stack web development skills.

## Local Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/purdue-research-matchmaker.git
```

Navigate to the project folder:

```bash
cd purdue-research-matchmaker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
